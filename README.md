# ZachetniyRepositoriyPila
Задача “Машина для очень веселой игры”
Необходимо разработать программно-аппаратное устройство для игры на базе Arduino Uno и p5.js. 

Алгоритм работы:
1.Пользователь надевает защитные очки и устанавливает голову в специальный держатель; кладет руки на клавиатуру.
2. На экране отображается инструкция, поле для ввода имени и кнопка СТАРТ для запуска игры. 
3. Пользователь вводит с помощью клавиатуры имя и нажимает на кнопку СТАРТ, после чего на экране появляется игровая сцена, на которой отображается текущее количество набранных очков и оставшееся время. 
4. Пользователь выполняет действия в соответствии с инструкцией в процессе игры. 
5. По истечении выделенного времени пользователь побеждает (а) или проигрывает (б), в зависимости от количества набранных очков. 
(a) Победа: на экран выводится победная анимация и текст, а пищалка издает победный писк. На экране также появляется кнопка “Таблица рекордов”, ведущая на соответствующую таблицу. Мандариновый диспенсер выдает один манадарин. 
(б) Поражение: на экран выводится пораженческая анимация и текст, а пищалка неприятно пищит. Насос начинает качать воду и производит полив (под напором, 3-5 секунд) лицевой части игрока, закрепленной в специальном держателе. Одновременно с поливом производится фотофиксация лица игрока с помощью вебкамеры в момент полива. Фото игрока выводится на экран. 

Функционал устройства:
* контроль уровня воды в резервуаре;
* оповещение о низком уровне воды и необходимости пополнить резервуар;
* контроль присутствия лица в лицевом держателе с помощью концевика;
* направленный полив под напором с помощью насоса;
* выдача мандарина с помощью сервопривода;
* пользовательский интерфейс (UI) на p5.js в браузере, который сообщается с Arduino посредством Serial;
*открытие/закрытие клапана с помощью микросервопривода;
* хранение таблицы рекордов в LocalStorage;
* фотофиксация лица игрока в момент полива.
